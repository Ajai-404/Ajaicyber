## Projects
Developed an IoT-Based Smart Medicine Scheduler to improve medication adherence for elderly patients through automated pill dispensing and real-time monitoring. As the Hardware Engineer on the project, I was responsible for assembling the smart pillbox hardware, integrating components such as the ESP32, servo motors, IR sensors, RTC module, LCD display, buzzer and GSM module, and supporting hardware testing and system integration. The system integrates Google Firebase, GSM SMS and Telegram notifications to provide reliable remote alerts for caregivers. It features automated dispensing, medication schedule management, adherence tracking using IR sensors and a web dashboard for monitoring and control. The project was developed using the Agile methodology and successfully tested through black-box and integration testing, demonstrating a reliable, user-friendly IoT healthcare solution.
